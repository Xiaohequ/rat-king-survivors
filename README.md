# 🐀 Rat King Survivors

A Vampire Survivors-like browser game built with HTML5 Canvas.

## How to Play
- **WASD** or **Arrow Keys** to move
- Control your Rat King and lead a swarm of rats
- Survive waves of enemies
- Collect upgrades every 15 seconds

## Features
- Swarm AI with 20+ rats
- Wave-based enemy spawning
- Upgrade system (Rats, Damage, Speed, etc.)
- Performance optimized for 100+ entities

## Play Online
👉 **https://rat-king-survivors.vercel.app**

## Run Locally
Simply open `index.html` in a browser.

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript
- No dependencies
